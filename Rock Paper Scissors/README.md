# Rock-Paper-Scissors Game Flowchart

This repository contains a flowchart outlining the logic for a two-player **Rock-Paper-Scissors** game.

---

## Game Logic

1. **Start**: Ensure 2 players are present, then begin a countdown.  
2. **Player Choices**: Players select **Rock**, **Paper**, or **Scissors** and reveal their choices.  
3. **Outcome**:
   - **Tie**: Both players choose the same object.  
   - **Winner**: Determined by standard rules:
     - Rock beats Scissors  
     - Scissors beat Paper  
     - Paper beats Rock.  
4. **End**: Declare the result and end the game.
